List of CSS [`filters`](https://developer.mozilla.org/en-US/docs/Web/CSS/filter) that can be used with the `filter` property. Use this functions to `setFilters` on the layer.

\{
  blur: blur(\$\{number}px),
  brightness: brightness(\$\{number}%),
  contrast: contrast(\$\{number}%),
  dropShadow: drop\-shadow(\$\{number}px  \| drop\-shadow(\$\{number}px  \| drop\-shadow(\$\{number}px  \| drop\-shadow(\$\{string}),
  grayscale: grayscale(\$\{number}%),
  hueRotate: hue\-rotate(\$\{number}deg),
  invert: invert(\$\{number}%),
  opacity: opacity(\$\{number}%),
  saturate: saturate(\$\{number}%),
  sepia: sepia(\$\{number}%)
}
