## About

LazyCanvas is a powerful and flexible library for creating and manipulating canvas elements in a lazy and efficient manner. It provides a simple and intuitive API for drawing shapes, images, text, and more, while optimizing performance by deferring rendering until necessary.

## Installation

You can install LazyCanvas via npm or yarn:

```bash
npm install lazycanvas
# or
yarn add lazycanvas
```
